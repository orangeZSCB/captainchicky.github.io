btcache.me is a free service for caching torrent metadata files online.

You can not search or list torrent files that are stored here, you can only access them if you already know the info_hash value of the torrent you want to download.
The way to access torrents is simple, you just use the url http://btcache.me/torrent/INFO_HASH_IN_HEX (note HEX values A-F must be in uppercase)
